blosc
=====

python meta-compression using snappy, used by blaze/BLZ/ContinuumIO
[web](http://blosc.pytables.org/)

python profiling
================

time modules: cProfile, line_profiler

memory modules: memory_profiler, guppy, **pympler**, heapy,
                massif (valgrind tool) (git://anongit.kde.org/massif-visualizer)

C++: gperftools

visualization: KCacheGrind, runsnakerun
